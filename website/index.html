产品设计

本地跑通，真正越来越懂你

默认零账单、零账号门槛；数据与身份留本机。可选云端与联网增强，但始终有纯本地路径。

  • 完全本地化

    对话、记忆、检索、执行可纯本地跑通；算力默认本机，可选 API 增强。

  • 真正易用

    一键安装、立即可用；主路径只有 la · la setup · la chat。

  • 长期多层次记忆

    Hot / Warm / Cold + Mem0：记住你，也懂得该记什么、何时介入。

  • 本机工具

    本地 Shell、写文件、工作区感知；执行前确认，危险命令拦截。

  • 本地 RAG

    文档进知识库，对话时可深度召回原文。

  • 日常三剑客

    la summarize · la news · la polish——总结、资讯、润色一键搞定。

快速开始

一行命令，装上就聊

需要 Python 3.10+ 与 pipx。当前版本 v0.4.0。 

pipx install "git+https://github.com/hezhenghui7338/localagent.git@v0.4.0"
la

有 API → 写入 Key；无 API → la setup -y 装 Ollama。

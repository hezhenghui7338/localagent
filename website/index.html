Product design

Fully local. Knows you long-term.

Zero bill and zero account by default; identity and data stay on your machine. Optional cloud and web — always with a pure local path.

  • Fully local

    Chat, memory, retrieval, and tools can run on-device; local compute by default, optional API boost.

  • Truly easy

    One-step install, ready to talk. Main path: la · la setup · la chat.

  • Layered long-term memory

    Hot / Warm / Cold + Mem0: remembers you — and knows what to keep, and when to step in.

  • On-device tools

    Local shell, file writes, workspace awareness; confirm before run, block dangerous commands.

  • Local RAG

    Ingest documents into a knowledge base; deep-recall source text in chat.

  • Daily essentials

    la summarize · la news · la polish — docs, briefing, and copy in one shot.

Quick start

One command, then chat

Needs Python 3.10+ and pipx. Current version v0.4.0. 

pipx install "git+https://github.com/hezhenghui7338/localagent.git@v0.4.0"
la

Have an API → set your key. No API → la setup -y for Ollama.

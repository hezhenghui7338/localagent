Product design

Local first. Really remembers you. Gets things done.

Install and you’re chatting, remembered, and getting work done — on-device by default, cloud optional; your identity and data stay on your machine.

  • Local First

    Zero-bill default path: chat, memory, retrieval, and tools run on-device. Main path: la · la setup · la chat. Optional cloud — identity and data stay local.

    See the zero-cost setup demo →

  • Memory Forever

    Hot / Warm / Cold + Mem0 across sessions. Remembers you — and knows what to keep, what to drop, and when to step in. Docs go into a local KB for deep recall.

    See the “remembers you” demo →

  • Actions Automated

    Shell, file writes, workspace; summarize · news · polish; scheduled brief. Confirm before side effects, block danger, show a receipt when done.

    See the deep-read demo →

Quick start

One command, then chat

Needs Python 3.10+ and pipx. Current version v0.4.0. 

pipx install "git+https://github.com/hezhenghui7338/localagent.git@v0.4.0"
la

Have an API → set your key. No API → la setup -y for Ollama.
